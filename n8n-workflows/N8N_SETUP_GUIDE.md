# HARAMOON n8n Workflow — Setup Guide

## What This Is

A permanent, self-contained n8n workflow that posts HARAMOON content to Instagram and Facebook. All brand rules, credentials, and post queue are baked directly into the workflow — you never need to re-explain anything to Manus or re-enter tokens.

---

## One-Time Import (5 minutes)

### Step 1 — Open Your n8n

Go to: **http://localhost:5678** in your browser

### Step 2 — Import the Workflow

1. Click **"Workflows"** in the left sidebar
2. Click **"Add Workflow"** (top right)
3. Click the **three-dot menu** (⋮) → **"Import from File"**
4. Select the file: `haramoon_ig_poster.json`
5. Click **Save** (top right)

### Step 3 — Activate It

Toggle the workflow to **Active** (top right switch).

That's it. The workflow is now permanently installed.

---

## How to Post

Every time you want to post the next item in the queue:

1. Open n8n → click the **HARAMOON workflow**
2. Click **"Execute Workflow"** (play button, top right)
3. Watch it run — you'll see green checkmarks on each node
4. The final node shows you: what posted, the Instagram URL, Facebook URL, and how many posts remain

**One click = one post.** It remembers what's been posted and never double-posts.

---

## When the Queue Runs Out

The last node will tell you: *"Queue is empty! Ask Manus to build a new batch."*

Open a new Manus chat and say:
> "Build me a new HARAMOON post batch for n8n. Give me the QUEUE array to paste into the Code node."

Manus will give you a new array. You paste it into the **"Load Queue & Brand Context"** node (the first Code node), replacing the old QUEUE array. Done.

---

## Adding Posts Manually

Open the **"Load Queue & Brand Context"** node and find the `QUEUE` array. Add a new object:

```json
{
  "id": "unique_id_here",
  "label": "Your Post Title",
  "type": "single",
  "images": ["https://your-image-cdn-url.jpg"],
  "caption": "Your caption text with hashtags\n\n#HARAMOON #KoreanSkincare"
}
```

For a carousel, set `"type": "carousel"` and add multiple URLs to `"images"`.

## Troubleshooting

| Problem | Fix |
|---|---|
| Red node — "Cannot read property 'id'" | Image URL is broken. Check the URL in the QUEUE array. |
| Red node — "OAuthException" | Token expired. Get a new token from Meta Business Suite and update CREDS in the Code node. |
| Queue shows 0 remaining but posts exist | Clear static data: Settings → Danger Zone → "Reset Static Data" |
| Facebook posts but Instagram doesn't | Check IG_USER_ID is correct (17841458399612807) |
